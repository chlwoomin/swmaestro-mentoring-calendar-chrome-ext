# 소마 달력 뷰어 크롬 익스텐션

AI·SW마에스트로 접수내역 페이지의 강의 일정을 달력 형식으로 보여주고, 구글 캘린더와 자동으로 동기화하는 크롬 익스텐션입니다.

## 기능

- 접수 완료된 강의를 달력 형식으로 시각화
- 날짜 클릭 시 해당 날 강의 목록·시간 표시
- **구글 캘린더 전체 추가** — OAuth 인증 후 모든 강의를 한 번에 등록
- **자동 동기화** — 페이지 방문 시 새 접수 강의는 자동 추가, 취소된 강의는 자동 삭제

## 설치 방법

### 1. 확장 프로그램 로드

1. 이 폴더를 원하는 위치에 저장합니다.
2. 크롬 주소창에 `chrome://extensions` 를 입력합니다.
3. 우측 상단의 **"개발자 모드"** 토글을 켭니다.
4. **"압축해제된 확장 프로그램을 로드합니다"** 버튼을 클릭합니다.
5. 이 폴더를 선택합니다.

### 2. 구글 캘린더 연동 설정 (전체 추가 / 자동 동기화 기능 사용 시)

1. [Google Cloud Console](https://console.cloud.google.com)에서 프로젝트를 생성합니다.
2. **API 및 서비스 → 라이브러리** 에서 **Google Calendar API** 를 사용 설정합니다.
3. **사용자 인증 정보 → OAuth 2.0 클라이언트 ID 만들기** 에서 유형을 **Chrome 앱** 으로 선택합니다.
4. `chrome://extensions` 에서 확인한 **확장 프로그램 ID** 를 애플리케이션 ID에 입력합니다.
5. 발급된 클라이언트 ID를 `manifest.json` 의 `oauth2.client_id` 에 붙여넣습니다.
6. **OAuth 동의 화면** 에서 본인 구글 계정을 테스트 사용자로 추가합니다.

## 사용 방법

1. `swmaestro.ai` 마이페이지 접수내역 페이지로 이동합니다.
2. 화면 우측 하단에 **파란 달력 버튼(FAB)** 이 나타납니다.
3. 버튼을 클릭하면 달력 팝업이 열립니다.
4. 날짜를 클릭하면 해당 날의 강의 목록과 시간이 표시됩니다.
5. 강의 제목을 클릭하면 상세 페이지로 이동합니다.

### 구글 캘린더 연동

| 버튼 | 설명 |
|------|------|
| 헤더 캘린더 **+** 아이콘 | 접수 완료된 전체 강의를 구글 캘린더에 일괄 추가 |
| 강의 카드 우측 캘린더 아이콘 | 해당 강의만 구글 캘린더 추가 페이지로 이동 |

- **전체 추가** 이후 페이지를 재방문하거나 **↻ 새로고침** 시, 새로 접수된 강의는 자동 추가되고 취소된 강의는 자동 삭제됩니다.
- 자동 동기화 결과는 FAB 버튼 위에 잠시 표시됩니다.
